| 显卡     | HD4000                   |
| 系统版本 | 10.15.7～12.2Beta1        |


## BIOS设置：
1. ACHI模式
2. CSM关闭
3. TV-D关闭
4. 升级BIOS版本（小白请勿操作）
5. 启用UEFI模式
6. 屏蔽独立显卡
